# ИИ-плагин для PyCharm
Гаходится в активной разработке

Расширение IDE PyCharm с интеграцией AI-ассистента для объяснения, рефакторинга и генерации кода. Реализовано на Kotlin с использованием IntelliJ Platform SDK.

> **Проект находится в стадии активной разработки.** Базовый функционал работает, оптимизируется стабильность и расширяется набор команд. Исходный код не публикуется до завершения публичного релиза.

---

## Технологический стек

| Категория | Технологии |
|:---|:---|
| **Язык** | Kotlin 1.9+ |
| **Framework** | IntelliJ Platform Plugin SDK, PyCharm Plugin API |
| **HTTP Client** | OkHttp3 (асинхронные запросы, Callback) |
| **JSON** | Gson (JsonArray, JsonObject, JsonParser) |
| **UI** | Swing (JTextPane, HTMLEditorKit, JPanel, BorderLayout) |
| **AI API** | OpenAI-compatible endpoint (gpt-4o-mini) |
| **Инструменты** | IntelliJ IDEA, Gradle, Git |

---

## Реализованный функционал (MVP)

### ✅ Чат-интерфейс с историей
- Двусторонний диалог: пользователь ↔ AI-ассистент
- Визуальное разделение сообщений (цветовые темы, временные метки)
- Автоматическая прокрутка к новым сообщениям
- Очистка истории («Новый чат»)

// Кнопка "Объяснить код" в действии:
1. Пользователь выделяет фрагмент кода в редакторе
2. Плагин извлекает выделенный текст через FileEditorManager

### Ключевые компоненты

**AIChatWindow (ToolWindowFactory)**
- Точка входа плагина, создание интерфейса
- Инициализация OkHttp3 клиента и истории чата

**Message Renderer (HTMLEditorKit)**
- Рендеринг сообщений через HTML-шаблоны
- Динамическое форматирование кода в <table> с стилями

**Code Extractor & Applier**
- FileEditorManager.getInstance(project).selectedTextEditor — получение выделенного кода
- WriteCommandAction — безопасная запись в документ (требование IntelliJ Platform)

**Chat History Manager (Gson)**
- Хранение диалога в JsonArray
- Сериализация/десериализация для отправки в API

**Async API Client (OkHttp3)**
- POST-запросы к OpenAI-compatible endpoint
- Обработка ошибок сети и некорректных ответов
