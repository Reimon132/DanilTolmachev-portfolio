# Анемометр (Система мониторинга ветра)

Профессиональный интерфейс для приёма, обработки и визуализации данных с ветровых датчиков. Интегрирован в судовые системы навигации и мониторинга окружающей среды. Проект прошёл тестирование, стабилизирован и готов к промышленной эксплуатации.

> Исходный код не публикуется по причине коммерческой конфиденциальности (NDA). Представлены только скриншоты интерфейса, архитектурное описание и технические решения, реализованные в финальной версии.

---

## Технологический стек

| Категория | Технологии |
|:---|:---|
| **Язык** | Python 3.10+ |
| **GUI Framework** | PyQt5 (QWidget, QPainter, QTimer, QThread) |
| **Графика** | Кастомная отрисовка (paintEvent), анимированные индикаторы, полярные диаграммы |
| **Протоколы** | Serial (`pyserial`), проприетарный протокол ветрового датчика |
| **БД** | SQLite (логирование показаний, история измерений, настройки калибровки) |
| **Платформы** | Windows / Linux (кроссплатформенная сборка) |
| **Архитектура** | Многопоточность, реактивный UI, модульное разделение компонентов |

---

## Ключевые особенности (реализовано в финальной версии)

- ✅ **Real-time визуализация**: направление и скорость ветра с частотой обновления 1–10 Гц
- ✅ **Полярная диаграмма**: круговая шкала направления ветра (0–360°) с плавной отрисовкой через QPainter
- ✅ **Исторические данные**: графики изменения скорости и направления ветра за выбранный период
- ✅ **Калибровка датчика**: интерфейс ввода поправочных коэффициентов с сохранением в SQLite
- ✅ **Пороговые предупреждения**: визуальная и звуковая сигнализация при превышении заданных значений
- ✅ **Экспорт отчётов**: выгрузка логов в CSV/Excel для последующего анализа
- ✅ **Кроссплатформенность**: единая кодовая база, адаптированная под Windows и Linux
- ✅ **Асинхронный приём данных**: выделенный поток для Serial-порта без блокировки основного UI

---


### Компоненты системы

**Serial Data Receiver**
- Непрерывный опрос COM-порта через `pyserial`
- Выделенный поток (`threading.Thread`) для изоляции I/O от UI
- Буферизация и парсинг входящих пакетов

**Data Parser**
- Преобразование сырых байтов/строк в физические величины
- Расчёт направления ветра (0–360°) и скорости (узлы/м/с/км/ч)
- Валидация диапазонов и фильтрация выбросов

**Calibration Module**
- Применение поправочных коэффициентов в реальном времени
- Сохранение/загрузка настроек через SQLite (`PermanentSetting`)
- Коррекция систематических погрешностей датчика

**UI Components**
- **Polar Compass Widget**: кастомная отрисовка направления ветра
- **Speed Gauge**: динамический индикатор скорости с цветовой градацией
- **History Plot**: временной график показаний на базе PyQtGraph/Matplotlib
- **Alert Panel**: модальные уведомления и индикаторы статуса

**SQLite Logger**
- Асинхронная запись измерений с временными метками
- Хранение истории за настраиваемый период
- Быстрая выборка для построения графиков
